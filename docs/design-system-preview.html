Design System Preview

Project Game, Candlelit War Desk

AI-native VTT and campaign operating system

Run the table like a command post, not a dashboard.

This direction treats the live session as the center of gravity. The battlemap owns the room. Transcript, approvals, recap, and world pressure sit in disciplined rails around it. The result feels dependable under pressure, atmospheric without costume-drama chrome, and distinct from every generic frosted-glass productivity clone.

Map-first live session
Approval-aware AI rail
Campaign HQ between sessions
Companion-grade mobile

Safe choices

  • Familiar VTT map center with clear combat strip
  • Productivity-style side rails for AI and transcript work
  • Status-forward degraded-mode language and diagnostics

Signature risks

  • Literary serif headlines instead of fantasy logo fonts
  • Warm brass and slate instead of purple gradients
  • Transcript elevated into a first-class live rail
